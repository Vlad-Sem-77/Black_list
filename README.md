# Реестр чёрных списков стран 🇷🇺

Мобильное веб-приложение (offline-first, один HTML-файл) для быстрого поиска: в каких регуляторных/санкционных перечнях РФ и РК находится та или иная страна, и что именно это означает.

## Что внутри

- **264 страны и территории** (по Общероссийскому классификатору стран мира, ОК 025-2001)
- **10 отслеживаемых перечней**:
  - Список ФАТФ (FATF)
  - Перечень стран, не выполняющих рекомендации ФАТФ (Росфинмониторинг)
  - Перечень автоматического обмена финансовой информацией (CRS, ФНС)
  - Перечень государств, не обеспечивающих обмен налоговой информацией (ФНС)
  - Перечень систематического неисполнения обмена страновыми отчётами (ФНС)
  - «Чёрный список» офшорных зон Минфина (налог на прибыль)
  - «Серый список» Минфина (временные послабления 2024–2026)
  - Офшорный список Банка России (для корреспондентских счетов)
  - Перечень недружественных государств (Распоряжение Правительства РФ № 430-р)
  - Классификация банковских рисков по странам (Казахстан: чёрный/серый/офшорный список + особое внимание)

## Как пользоваться

1. Открыть `index.html` в любом браузере (Chrome, Safari, Firefox — неважно, ПК или телефон).
2. Найти страну через поиск (ищет по началу слова в названии).
3. Открыть карточку страны → увидеть, в каких перечнях она числится.
4. Открыть карточку конкретного перечня → краткое описание, реквизиты НПА для этой страны, и кнопка **«Полный текст НПА и разбор перечня»** — с полным текстом первоисточника, датами вступления в силу, числом стран и последствиями.
5. На Android/мобильных — кнопка/жест «назад» листает экраны внутри приложения, а не закрывает его.

## Технологии

Просто HTML + CSS + vanilla JavaScript. Никакого бэкенда, никаких серверов и баз данных — все данные (264 страны × 10 перечней) встроены прямо в файл в виде JSON. Работает офлайн (кроме шрифтов Google Fonts при первом открытии).

## Обновление данных

Приложение специально спроектировано так, чтобы **движок (логику, вёрстку, поиск) не нужно было трогать при обновлении данных**. Все тексты и списки стран лежат одним JSON-блоком в начале `<script>` — начале файла `index.html` (`const DATA = {...}`). Чтобы обновить:

1. Актуализируйте исходную таблицу (Excel/CSV) со странами и перечнями.
2. Пересоберите JSON в том же формате (см. структуру ниже).
3. Замените блок `const DATA = {...}` в `index.html`.

### Структура данных

```json
{
  "lists": [
    {
      "id": "unfriendly",
      "short": "Недружественные страны",
      "full": "Перечень недружественных иностранных государств и территорий",
      "desc": "Краткое описание перечня...",
      "full_text": "Полный текст НПА и разбор последствий..."
    }
  ],
  "countries": [
    {
      "n": "ГЕРМАНИЯ",
      "c": "DE │ DEU │ 276",
      "l": [["unfriendly", "Распоряжение Правительства РФ от 05.03.2022 № 430-р"]]
    }
  ]
}
```

## Дисклеймер

Приложение носит справочный характер и не является юридической консультацией. Данные приведены на основании открытых нормативных правовых актов по состоянию на момент составления исходной таблицы — актуальность и полноту рекомендуется дополнительно сверять с первоисточниками (сайты ФНС, Минфина, Банка России, Правительства РФ, ФАТФ).
